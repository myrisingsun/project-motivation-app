# Проектная мотивация

HR-инструмент для расчёта проектных бонусов сотрудников на основе участия в проектах, оценок ABC и выполнения контрольных точек.

## Возможности

- **Два режима расчёта** — мультипликаторный и компонентный
- **ABC-рейтинг** сотрудников с автоматическим определением новичков
- **Контрольные точки** проектов с весами по методологии PMI
- **Ограничение проектов** на сотрудника (по умолчанию — не более 3)
- **График выплат** — квартальный (Q1–Q4) или годовой
- **Импорт** сотрудников из Excel, **экспорт** полного отчёта в Excel
- **Дашборд** с графиками распределения фонда и загрузки
- **Снимки состояния** — версионирование данных
- Авто-сохранение в localStorage, без сервера

## Быстрый старт

```bash
npm install
npm run dev
```

Откроется на [http://localhost:5173](http://localhost:5173).

Для доступа из локальной сети сервер уже настроен на `0.0.0.0`:

```bash
npm run dev   # также доступен по IP в LAN
```

## Сборка

```bash
npm run build   # → dist/
npm run preview # превью продакшн-сборки
```

Папку `dist/` разместить на Vercel, Netlify, GitHub Pages или любом статическом хостинге.

## Формулы расчёта

**Мультипликаторный режим:**
```
Потолок = Оклад × Мультипликатор   (для роли S: Потолок / 3)
Бонус   = Потолок × Вес_вехи × (Факт_дни / План_дни) × Коэфф_ABC
```

**Компонентный режим:**
```
Бонус = Потолок × (Вехи×W1 + БюджетШкала×W2 + ABC×W3) / 100
```
где W1 + W2 + W3 = 100 (по умолчанию 40 / 40 / 20).
