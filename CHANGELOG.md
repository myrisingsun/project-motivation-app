# Changelog

## [2.0.0] — 2026-03-26 — Спринт 2

### Новые возможности

#### Команда проекта (ProjectsPage)
- Секция «Команда проекта» в каждой карточке проекта
- Отображение текущих участников в виде чипов с именами
- Добавление исполнителя через выпадающий список (только незадействованные сотрудники)
- Удаление исполнителя с подтверждением через AlertDialog — удаляет все записи участия по сотруднику в проекте с автоматическим пересчётом

#### Тёмная тема (App)
- Кнопка Sun/Moon в хедере — переключает тёмную/светлую тему
- Состояние сохраняется в localStorage (`pm-calc-theme`)
- Полная поддержка тёмного режима через CSS-переменные coss ui

#### Валидация участия (ParticipationPage)
- Незаполненные строки (нет сотрудника или КТ) подсвечиваются жёлтым
- Alert с подсчётом незаполненных записей — они не учитываются в расчёте

### Исправления

#### Баг: агрегация периодов H1/H2 в графике выплат (ResultsPage)
- Ранее записи с периодом `H1`/`H2` пересчитывались заново при переключении в «Полугодично»
- Теперь: если период уже `H*`, используется как есть; `Q1/Q2 → H1`, `Q3/Q4 → H2`

#### Баг: цифры вместо имён в столбцах «Сотрудник» и «Контрольная точка» (ParticipationPage)
- Base UI Select некорректно сравнивал числовые ID при смешанных типах (number/string)
- Поля «Сотрудник» и «Контрольная точка» переведены на нативный `<select>` — надёжное сопоставление числовых ID
- Добавлена нормализация ID при загрузке из localStorage (`+r.empId`, `+r.checkpointId`)
- Расчёты (`calculations.js`) используют нестрогое сравнение `==` для поиска по ID

### Стилизация и инфраструктура

#### CSS-тема coss ui (index.css)
- Добавлены все семантические токены coss из официального registry (`coss.com/ui/r/style.json`)
- `--primary`, `--background`, `--foreground`, `--border`, `--input`, `--ring`, `--muted`, `--accent`, `--popover`, `--card`, `--radius-*` и полный набор sidebar-переменных
- `@layer base` с `border-border`, `bg-background`, `text-foreground` на body
- Светлая и тёмная тема полностью согласованы с дизайном coss.com/ui

#### Хедер и навигация (App)
- Переведены на семантические токены: `bg-background`, `border-border`, `text-foreground`, `text-muted-foreground`
- Версия обновлена до `v2.0`

## [1.0.0] — 2026-03-25 — Спринт 1

### Базовая архитектура HR-калькулятора мотивации

- React 18 + Vite 6 + Tailwind CSS v4
- coss ui компоненты: Tabs, Dialog, AlertDialog, Toast, Tooltip, Badge, Alert, Card, Accordion, NumberField, RadioGroup, Switch, Fieldset, Separator
- 6 страниц: Dashboard, Employees, Projects, Participation, Results, Settings
- Два режима формулы: множительный и компонентный
- Импорт сотрудников из Excel (SheetJS), экспорт отчёта
- Расчёт премий с ABC-грейдами, новичками, лимитами проектов
- Автосохранение в localStorage (500 мс debounce)
- Recharts: PieChart, BarChart на Dashboard
- Печать через `window.print()` + `@media print`
